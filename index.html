It Gets Conflicated

Did I just use a fake word? Yes.
Should it be a real word? Also yes.

When something is conflicated it means that something is confusing because it's complicated.
And the way the Nash Equilibrium can represent trust is conflicated.

But it's my job to make it simple for you.

Essentially trust needs three things.
One, trust needs to have repeated interactions. Two, both parties need to be able to win. Three, there needs to be a low number of miscommunications.

Simple aint it?
If any one of these three is missing, then the trust breaks down.

Every game you've tried so far has been fair.
It was only a difference in strategies that resulted in different outcomes.
